## TwineHacker

<https://github.com/La-Li-Lu-Le-Lo/TwineHacker> (itself based on a tool by
@spectr3.9911)

TwineHacker pioneered the "F12 DevTools panel for editing SugarCube variables"
concept that inspired this project's DevTools panel. TwineHacker does not
publish a license, so Twine Peeks aims to include **no code or assets from
it** — pre-release private builds temporarily used its icons as placeholders,
and those were replaced with original artwork in v1.7.0. If you believe any
TwineHacker-derived code remains in this repository, please open an issue and
it will be rewritten or removed.

## hituro's CHATSYSTEM

<https://github.com/hituro/hituro-makes-macros>

The Chat tab exists to inspect games built with hituro's CHATSYSTEM macros.
No CHATSYSTEM code is included; the tab reads game state at runtime.
